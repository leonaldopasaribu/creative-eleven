Relative Layout and Linear Layout in Android Studio

  • Yepta Zagarino Samosir

  • April 18, 2020

In my first article, I will discuss layouts that are often used in android programming, namely Relative Layout and Linear Layout. Previously, we will discuss what is the layout. The layout is a display that regulates the layout of the components that we will show to users. The components in question are like pictures, text, or other components. On Android Studio, processing layout views are created in .xml format.

What is the Relative Layout and Linear Layout? What's the difference between the two layouts?

  1. Relative Layout

    2. What is the relative layout? Relative Layout is a layout arranged by displaying a widget inside it like a layer. Relative Layout is a relatively free layout so that widget can be arranged freely. A widget can be above or below another widget. So it can be interpreted that Relative Layout is the design of the appearance of the application with the layout of objects or components freely.

    (Relative Layout|Source image: CodePolitan.com)

  2. Linear Layout

    3. What is a Linear Layout? Linear Layout is a layout in which components are arranged in one direction, namely horizontal or vertical. Linear Layout makes it easy to use but limits its flexibility. For applications that have a less complex design, the linear layout is very suitable for use.

    (Linear Layout|Source image: CodePolitan.com)

    The two types of layouts above are the layouts that are most often used to learn early about layouts on Android. There are several other layouts that are used to obtain a better appearance and I will discuss in the next article. Thanks.
